nutze beigefügte liste mit rezepten als input und stell wie folgt eine exportierbare liste zusammen.

Aufbau wie folgt (Spalten):

Gericht / Zutat / Kategorie / typ / menge / einheit

Erläuterungen:

gericht: rezeptname
zutat: liste mit zutaten
kategorie der jeweiligen zutat: fleisch / gemüse / teigwaren / brot / getränke / milchwaren / saucen/konserven / vegi / getränke
typ: immer "gericht"
menge: menge der jeweiligen zutat
einheit: einheit der menge der jeweiligen zutat
